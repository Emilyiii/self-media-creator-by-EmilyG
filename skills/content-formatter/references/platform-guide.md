# Platform Formatting Guide

## 公众号 (WeChat) Guidelines

### Visual Hierarchy

**Title (H1)**
- Only one per article
- Largest, most prominent
- Center-aligned in preview

**Section Headers (H2)**
- Use `##` in Markdown
- Every 2-4 paragraphs
- Keep readers oriented

**Subsection Headers (H3)**
- Use `###` for nested structure
- Optional, use sparingly

### Formatting Best Practices

**Emphasis**
- `**bold**` for key terms and important points
- `*italic*` for quotes or subtle emphasis
- `> blockquote` for important callouts

**Lists**
- Use `-` or `*` for unordered lists
- Use `1.` for ordered steps
- Keep list items parallel in structure

**Code**
- Inline code: `` `code` ``
- Code blocks: triple backticks with language

**Images**
- Cover: 900×383px (2.35:1) or 900×500px
- Content: 900px width, flexible height
- Max 10MB per image
- Format: JPG, PNG, GIF

### Spacing Rules

- One blank line between paragraphs
- Two blank lines before headers
- One blank line after images
- Horizontal rule `---` for major section breaks

### Mobile Optimization

- Keep paragraphs under 4 lines on mobile
- Use short sentences
- Break up long lists
- Ensure images are readable on small screens

## 小红书 (Xiaohongshu) Guidelines

### Content Structure

**Hook (First 2 lines)**
- Must grab attention immediately
- Visible in feed without expanding
- Use emoji + compelling statement

**Body**
- Mobile-first: short paragraphs
- Bullet points over long text
- Emojis as visual anchors

**Closing**
- Personal touch
- Call to action (comment, save, follow)
- Hashtags

### Formatting Patterns

**Bullet Points**
```
📌 重点：
• 第一点简洁明了
• 第二点补充说明
• 第三点总结升华
```

**Numbered Lists**
```
✨ 三个技巧：
1️⃣ 第一个技巧
2️⃣ 第二个技巧
3️⃣ 第三个技巧
```

**Comparison**
```
❌ 错误做法
• 长篇大论没重点
• 专业术语堆砌

✅ 正确做法
• 分段清晰易读
• 口语化表达
```

### Hashtag Strategy

**Formula:**
- 1-2 大流量标签 (#干货分享 #经验分享)
- 2-3 垂直领域标签 (#AI工具 #效率神器)
- 1-2 细分标签 (#ChatGPT技巧 #职场新人)

**Placement:**
- Always at the end
- Space-separated
- No more than 5-6 tags

### Length Guidelines

| Element | Recommended Length |
|---------|-------------------|
| Hook | 10-20 characters |
| Bullet points | 15-30 characters each |
| Total post | 200-500 characters |
| Hashtags | 3-5 tags |

### Tone by Style

**Casual (休闲)**
- 口语化，像和朋友聊天
- 多用"我"、"咱们"
- 表情符号丰富
- 可以适当用网络用语

**Professional (专业)**
- 简洁有力
- 数据支撑
- 结构清晰
- 适度emoji

**Friendly (亲切)**
- 温暖治愈
- 分享感强
- 鼓励互动
- 情感共鸣

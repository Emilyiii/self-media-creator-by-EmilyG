# Image Generation Prompt Guide

## Prompt Structure

Effective prompts follow this structure:

```
[主体] + [场景/环境] + [风格] + [构图] + [光影/氛围] + [技术参数]
```

## Examples by Content Type

### 公众号封面图

**通用公式：**
```
[主题关键词] + 扁平插画风格 + 明亮配色 + 简洁构图 + 适合公众号封面
```

**示例：**
```
AI助手帮助人类工作的场景，扁平插画风格，蓝色和橙色配色，简洁现代，人物和机器人协作，明亮背景，适合科技公众号封面，16:9比例
```

### 概念解释图

**通用公式：**
```
[抽象概念可视化] + 信息图表风格 + 清晰层次 + 图标元素
```

**示例：**
```
机器学习流程可视化，数据输入到输出的流程图，扁平设计，蓝色渐变，图标化元素，白色背景，清晰易读，适合技术文章配图
```

### 工具对比图

**通用公式：**
```
[工具A] vs [工具B] + 分屏对比布局 + 各自特色视觉化
```

**示例：**
```
两个AI工具对比，左右分屏布局，左侧蓝色调代表工具A，右侧橙色调代表工具B，各自展示核心功能图标，扁平插画风格，简洁现代，适合工具评测文章
```

## Style Keywords

### 插画风格
- 扁平插画 (flat illustration)
- 2.5D等距视角 (isometric)
- 手绘风格 (hand-drawn)
- 孟菲斯风格 (Memphis style)
- 新拟态 (neumorphism)
- 玻璃拟态 (glassmorphism)

### 艺术风格
- 赛博朋克 (cyberpunk)
- 蒸汽波 (vaporwave)
- 极简主义 (minimalist)
- 复古风 (retro/vintage)
- 未来主义 (futuristic)

### 技术参数
- 比例: 16:9, 1:1, 9:16, 4:3
- 质量: high quality, detailed, 8k
- 渲染: 3D render, octane render, unreal engine

## Anti-Patterns (Avoid)

1. **过于复杂的场景** - 会导致画面混乱
2. **多个人物特写** - AI生成人物面部容易崩
3. **需要精确文字** - 除非使用专门支持文字的模型
4. **过于抽象的描述** - 需要具体视觉元素

2. 支持16:9、1:1、9:16等常见比例
3. 对"扁平插画"、"国潮风"等中文风格词理解较好

# Visual Style Reference

## Style Catalog

### 1. 扁平插画 (Flat Illustration)
**Best for:** 公众号配图、概念解释、工具介绍

**Characteristics:**
- 简洁几何形状
- 纯色或渐变填充
- 无描边或细描边
- 阴影简单或没有

**Prompt keywords:**
- flat illustration
- vector art
- minimal design
- clean lines

**Example scenes:**
- 人物工作场景
- 概念可视化
- 产品界面展示

---

### 2. 2.5D等距视角 (Isometric)
**Best for:** 流程图、架构图、产品展示

**Characteristics:**
- 45度视角
- 三维感但保持平面化
- 适合展示层次结构

**Prompt keywords:**
- isometric view
- 2.5D illustration
- axonometric

**Example scenes:**
- 系统架构图
- 工作流程展示
- 房间/空间布局

---

### 3. 3D渲染 (3D Render)
**Best for:** 封面图、产品展示、视觉冲击

**Characteristics:**
- 真实材质感
- 光影效果丰富
- 立体感强

**Prompt keywords:**
- 3D render
- octane render
- blender 3D
- cinematic lighting

**Example scenes:**
- 科技产品展示
- 抽象概念立体化
- 场景氛围图

---

### 4. 手绘风格 (Hand-drawn)
**Best for:** 轻松话题、个人博客、教育内容

**Characteristics:**
- 不规则线条
- 有机形状
- 温暖亲切感

**Prompt keywords:**
- hand-drawn
- sketch style
- doodle art
- watercolor

**Example scenes:**
- 笔记式图解
- 思维导图
- 生活场景

---

### 5. 新拟态 (Neumorphism)
**Best for:** UI设计展示、按钮/卡片展示

**Characteristics:**
- 柔和的阴影
- 凸起/凹陷效果
- 单色或近色系

**Prompt keywords:**
- neumorphism
- soft UI
- subtle shadows

**Example scenes:**
- 界面元素展示
- 开关/按钮设计
- 卡片式布局

---

## Style Selection Guide

| Content Type | Recommended Style | Why |
|--------------|-------------------|-----|
| 科技产品介绍 | 3D渲染/扁平插画 | 专业感/现代感 |
| 教程/指南 | 扁平插画/手绘 | 易读/亲切 |
| 数据分析 | 2.5D等距 | 清晰展示层次 |
| 概念解释 | 扁平插画 | 简洁易懂 |
| 工具对比 | 分屏+扁平 | 清晰对比 |
| 个人感悟 | 手绘/水彩 | 情感共鸣 |

## Composition Patterns

### 居中构图
- 主体在画面中央
- 适合：封面图、概念图
- 稳定、正式

### 三分法构图
- 主体在画面1/3处
- 适合：场景图、叙事性图片
- 动态、有层次

### 对称构图
- 左右或上下对称
- 适合：对比图、平衡感
- 稳定、对比强烈

### 引导线构图
- 利用线条引导视线
- 适合：流程图、路径展示
- 有方向感、引导阅读
